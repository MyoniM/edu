# edu

edu is a minimal, transpiled programming language. edu is transpiled into javascript. <br/>
**edu lets you write code in amharic!!!**

## Features <br/>
- null
- boolean
- numbers
- strings
- array
- variables
- functions
- lambda functions
- comments
